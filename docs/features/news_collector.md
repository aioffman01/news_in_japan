# 기능 정의서 - 일본 건축 뉴스 수집기 (NewsInJapan)

## 1. 개요
일본 5대 신문사(요미우리, 아사히, 마이니치, 니혼게이자이, 산케이)에서 발행되는 건축/건설 관련 소식을 RSS 피드로 수집하여 필터링하고, Gemini API를 통해 한국어 제목 및 핵심 요약(줄거리)으로 가공하여 로컬 SQLite DB에 저장 및 제공하는 시스템입니다.

## 2. 주요 기능 및 요구사항
- **뉴스 피드 수집**:
  - 일본 5대 신문사(Yomiuri, Asahi, Mainichi, Nikkei, Sankei)의 RSS 피드 파싱.
  - 전날(어제) 발행된 기사들을 대상으로 필터링.
  - '建築', '建設', '都市開発', '住宅', '不動産', 'インフラ' 등 건축 및 건설 관련 주요 키워드 일치 여부 판별.
- **번역 및 요약 (Gemini API)**:
  - 일본어 원문 제목과 본문(또는 RSS Description)을 기반으로 Gemini API에 요약 요청.
  - 결과값: 한국어 번역된 제목 + 2~3문장의 한국어 요약문.
- **데이터 적재**:
  - SQLite DB (`news.db`) 테이블 구조에 고유 식별값(URL 해시 등)을 이용해 중복 제거 후 최대 10개 내외의 주요 뉴스 적재.
- **API 서버**:
  - `/api/v1/news`: DB에 적재된 가공 완료 뉴스 리스트 조회 API.
  - `/api/v1/news/collect` (POST): 수집 프로세스를 강제 트리거하는 API.
- **컨테이너화**:
  - Docker Compose를 통해 Backend 및 Frontend 기본 구조를 손쉽게 배포할 수 있는 환경 제공.
